利用GitHub搭建个人博客
步骤：
  1. 安装Mac系统的包管理工具 homebrew
  2. 通过homebrew安装nodejs
  3. 安装hexo

安装homebrew
>ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"  

安装nodejs
```
brew install node  
```

安装hexo
```
sudo npm install -g hexo
```

安装好工具后，接下来开始搭建博客
